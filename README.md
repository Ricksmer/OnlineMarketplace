# OnlineMarketplace

## Overview
OnlineMarketplace is a full-stack educational e-commerce platform designed to simulate a real-world online marketplace. It allows users to browse products, while sellers can manage listings, addresses, and vouchers. The project focuses on practicing end-to-end web development.

## Features
- User authentication with session handling
- Role-based access (seller validation)
- Seller dashboard interface
- Add and manage products
- Add address information
- Create and manage vouchers

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## Purpose
This project is developed to strengthen understanding of:
- Client-server interaction
- Database design
- Session management
- Full-stack development
